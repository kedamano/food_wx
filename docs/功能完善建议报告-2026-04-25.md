## 二、后端未完成 / 伪实现功能

### 2.1 🔴 高优先级（影响核心流程）

#### 1. 微信登录注册 — openid 模拟
- **位置**：`UserController.wechatRegister()`，第 221 行
- **问题**：`String openid = "wx_" + code.substring(0, 28);` — 用 code 字符串拼接伪造 openid，未接入微信官方 `code2session` 接口
- **影响**：微信注册渠道完全失效，openid 无法唯一标识用户
- **建议**：
  ```java
  // 调用 https://api.weixin.qq.com/sns/jscode2session
  // 参数：appid + secret + js_code + grant_type=authorization_code
  // 返回：openid + session_key
  String openid = wechatService.getOpenId(code); // 需实现 WechatService
  ```
- **依赖**：需要在 `application.yml` 中配置 `wx.appid` 和 `wx.secret`

#### 2. 注册接口 — 验证码未验证
- **位置**：`UserController.register()`，第 243 行；`UserService.register()`
- **问题**：`register()` 接收 `User` 对象直接注册，虽然前端传了 `verifyCode` 字段，但后端完全未验证（仅 `wechat-register` 独立发出，普通注册未走 `/user/verify-code`）
- **建议**：在 `UserService.register()` 中调用 `VerifyCodeUtil.verifyCode(phone, code)` 并在成功后清除

#### 3. 订单支付 — 无真实支付接入
- **位置**：`OrderController.payOrder()` → `OrderService.payOrder()`
- **问题**：仅更新状态为"已支付"，未接入微信支付 API（JSAPI/小程序支付）
- **影响**：用户无法完成真实资金流转
- **建议**：
  1. 集成微信支付 `unifyed_order` 接口，生成预支付订单
  2. 返回 `prepay_id` 及签名给前端
  3. 前端调用 `wx.requestPayment()` 完成支付
  4. 后端实现支付结果回调（`/pay/notify`），在回调中修改订单状态

#### 4. 积分日志 — 未持久化到数据库
- **位置**：`UserController.updateUserPoints()`，第 418 行
- **问题**：`System.out.println("[积分变动] ...")` — 仅打印日志，未写入数据库
- **影响**：`pages/points-history` 页面无法显示真实积分变动记录，前端当前用 `pointsHistory` 本地存储模拟
- **建议**：
  1. 新建 `PointsLog` 实体和 `points_log` 表（字段：id, userId, change, type, reason, createTime）
  2. 所有积分变动（签到、下单完成、管理员操作）均写入该表
  3. 新增 `GET /user/points/history/{userId}` 接口供前端调用

#### 5. 签到记录 — 存在内存，重启即失
- **位置**：`SignInUtil.java`
- **问题**：`Map<Integer, LocalDate> signInRecords` 和 `Map<Integer, Integer> consecutiveDays` 均为 `static` 内存变量，服务重启后全部丢失
- **影响**：用户今天签到，服务重启后明天可重复签到拿积分
- **建议**：
  1. 新建 `SignInRecord` 实体和 `sign_in_records` 表
  2. 或直接复用 `points_log` 表，通过 `type=SIGN_IN` 判断当日是否已签到

### Haversine 距离计算（Java）
```java
public static double calculateDistance(double lat1, double lng1, double lat2, double lng2) {
    final int R = 6371; // 地球半径 km
    double dLat = Math.toRadians(lat2 - lat1);
    double dLng = Math.toRadians(lng2 - lng1);
    double a = Math.sin(dLat/2) * Math.sin(dLat/2)
             + Math.cos(Math.toRadians(lat1)) * Math.cos(Math.toRadians(lat2))
             * Math.sin(dLng/2) * Math.sin(dLng/2);
    return R * 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
}
```
